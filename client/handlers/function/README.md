1. liquidline_frame.py - 底层SDK封装
输入:
相机连接信息: IP、端口、用户名、密码
通道号、Y坐标位置(0-576)、液位线文本

输出:
布尔值(操作成功/失败)
在相机画面上叠加显示液位线字符

功能:
初始化海康威视SDK
登录相机设备
使用SDK的字符叠加功能在指定Y坐标绘制液位线
清除液位线显示
资源清理和注销
2. liquidline_frame.py- 业务逻辑层
输入:
RTSP地址
检测结果字典:
{
    'level_y': int,        # 液位线Y坐标
    'level_value': float   # 液位值(米)
}

输出:
布尔值(更新成功/失败)
在相机画面上动态更新液位线位置和数值
功能:
DynamicLevelLine: 管理液位线的动态更新,支持线程安全操作
LevelLineController: 与检测系统集成,解析RTSP地址,根据检测结果自动更新液位线
支持显示液位数值(如"━━━━ 液位: 10.50m ━━━━")
工作流程:
解析RTSP地址获取相机连接参数
启动服务并登录相机
接收检测系统的液位数据
动态更新相机画面上的液位线位置和数值
