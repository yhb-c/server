如果已经设置成功，则不重复设置
api服务启动
export LD_LIBRARY_PATH=/home/lqj/liquid/server/lib/lib:$LD_LIBRARY_PATH && cd /home/lqj/liquid/server/network/api && nohup ./liquid-api > /home/lqj/liquid/logs/api.log 2>&1 &
使用systemd将API和WebSocket配置为系统服务，开机自动启动
查看状态: sudo systemctl status liquid-api 或 sudo systemctl status liquid-websocket
停止服务: sudo systemctl stop liquid-api 或 sudo systemctl stop liquid-websocket
重启服务: sudo systemctl restart liquid-api 或 sudo systemctl restart liquid-websocket
查看日志: sudo journalctl -u liquid-api -f 或 sudo journalctl -u liquid-websocket -f
