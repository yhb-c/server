# SSH免密连接功能说明

## 功能概述

本功能为液位检测系统客户端添加了SSH免密连接管理功能，可以自动配置和管理到远程服务器(192.168.0.121)的SSH连接，并支持远程服务的启动和管理。

## 主要特性

1. **自动SSH配置**: 自动生成SSH密钥对并配置免密登录
2. **网络连接检测**: 启动前检测到服务器的网络连接
3. **远程服务管理**: 检查和启动远程Go API服务和Python推理服务
4. **服务状态监控**: 实时监控本地和远程服务状态
5. **智能服务启动**: 根据服务状态智能选择启动本地或使用远程服务

## 文件说明

- `ssh_manager.py`: SSH连接管理器，负责SSH配置和远程命令执行
- `main.py`: 主程序，集成了SSH连接功能
- `test/test_ssh_connection.py`: SSH连接测试脚本
- `logs/`: 日志目录，存放系统运行日志

## 使用方法

### 自动使用（推荐）

直接运行主程序，系统会自动检查和配置SSH连接：

```bash
python main.py
```

### 手动测试SSH连接

运行测试脚本验证SSH连接功能：

```bash
python test/test_ssh_connection.py
```

## 工作流程

1. **启动检查**: 系统启动时自动检查SSH配置状态
2. **网络测试**: 测试到服务器192.168.0.121的网络连接
3. **SSH配置**: 如果未配置，自动生成密钥对并设置免密登录
4. **远程服务检查**: 通过SSH检查远程服务运行状态
5. **服务启动**: 根据需要启动远程服务
6. **本地服务**: 启动本地服务作为备用
7. **状态监控**: 持续监控所有服务状态

## 故障排除

### SSH连接失败
1. 检查网络连接到192.168.0.121
2. 确认服务器SSH服务正在运行
3. 检查防火墙设置
4. 手动运行测试脚本诊断问题

### 远程服务启动失败
1. 检查服务器上的conda环境配置
2. 确认项目文件完整性
3. 检查服务器资源使用情况
4. 查看远程日志文件

### 权限问题
1. 确认SSH密钥权限正确(600)
2. 检查服务器用户权限
3. 确认项目目录访问权限

## 安全说明

- SSH密钥存储在用户主目录的.ssh文件夹中
- 使用RSA 4096位密钥加密
- 配置了连接保活机制
- 禁用了主机密钥检查（仅限内网使用）
